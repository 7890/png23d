Usage

Although the tool was primarily developed for 3D printing applications the STL output is two maifold and most CAD applications will import the output correctly.

Usage is based around the commandline in a similar manner to the ImageMagick two dimensional image manipulation tool.

The input file must be a PNG image that the libpng library can read and the output file may have any arbitrary filename. The output type is controlled by the -o switch. The full set of available command line switches may be found in the man page provided.
