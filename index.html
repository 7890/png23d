png23d by kyllikki

PNG23D a 3d mesh generation tool

This tool converts images in the PNG image format into a 3 dimensional representation. Two output formats (OpenSCAD and STL) are currently supported with several options for output parameters.

Dependencies

libpng

Install

should be as simple as installing the dependencies and typing make install, if you want man pages make install-man too.
